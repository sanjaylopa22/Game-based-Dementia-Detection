About Project

Enhancing Cognitive Health and Predicting Dementia

This project is an innovative web-based application designed to aid in the early detection of dementia through interactive cognitive games and advanced deep learning models. Recognizing the critical importance of early diagnosis in managing dementia, our solution combines engaging gameplay with sophisticated predictive analytics to provide a comprehensive and accessible screening tool.

Motivation and Objective: The primary motivation behind this project is the growing need for early and accurate dementia diagnosis, which can significantly improve patient care and outcomes. Our objective is to create a user-friendly platform that not only makes the screening process engaging but also leverages modern technology to enhance diagnostic accuracy.

How It Works: The application features two main levels of a memory card matching game, each designed to assess different cognitive abilities.
Our project features an innovative approach to early dementia detection through engaging memory games and advanced deep learning models. In Game Level 1: Basic Match, users participate in a classic memory card matching game. A countdown timer assesses short-term memory by giving users a brief period to memorize card positions before they begin matching pairs. Performance is tracked based on the user's ability to complete the task within a set number of clicks or a specified time limit. Game Level 2: Ultimate Test introduces increased complexity with dynamic grid changes and hidden bombs that reset the game if clicked. Users must complete this level within a click threshold or time limit to proceed. Upon completion, they are redirected to a prediction page where they input health metrics and provide a facial image for further analysis. Our predictive system employs two convolutional neural network models: the MOD-1D-CNN, which processes health metrics such as blood oxygen levels, weight, age, diabetes status, body temperature, and heart rate; and the MOD-2D-CNN, which analyzes facial images to detect signs of dementia based on facial features. Together, these models offer a comprehensive assessment of dementia risk.

Future Scope: We aim to enhance the application by integrating real-time video analysis to monitor users' behavior and further refine the predictive accuracy. The project also plans to transition to a Flask-based architecture for better integration with robust databases and cloud platforms, ensuring scalability and reliability.

Game Levels

Level - 1 (Basic Match)

Welcome to Level 1, where we lay the foundation for your memory-enhancing journey. Your task is simple: match identical cards by clicking on them. Follow these steps to play successfully:

  • Begin by clicking on any card to reveal its image.
  • Take a moment to remember the picture and its specific position on the grid.
  • Your goal is to find and click on the card that shares the same image and position.
  • Continue this process, pairing up cards until you've successfully cleared the entire board.

level - 2 (Ultimate Test)

You've reached the grand finale, Level 2, which combines all the challenges you've faced thus far. Achieve victory with these comprehensive instructions:

  • Begin by clicking on any card to unveil its image.
  • Commit both the picture and its initial location to memory.
  • Proceed to match cards, navigate the grids, adapt to rotations, and anticipate grid swaps.
  • To emerge victorious in this ultimate memory marathon, you must harness all your skills and cognitive abilities.

FAQs

+

What is the purpose of this project?

The purpose of this project is to create an engaging and accessible tool for the early detection of dementia. By combining interactive memory games with advanced machine learning models, we aim to provide a comprehensive screening platform that can help identify early signs of dementia and facilitate timely medical intervention.

+

How do the memory games work?

Our application features two levels of memory card matching games. In Level 1: Basic Match, users are given a grid of cards to memorize and match within a certain number of clicks or time limit. Level 2: Ultimate Test introduces additional complexities, such as dynamic grid changes and hidden bombs, to further challenge users' cognitive abilities. Performance in these games helps assess cognitive function.

+

What types of data are used to predict dementia?

The predictive models in our application use two types of data. The MOD-1D-CNN model analyzes health metrics such as blood oxygen levels, weight, age, diabetes status, body temperature, and heart rate. The MOD-2D-CNN model evaluates facial images to detect signs of dementia based on facial features. Together, these models provide a comprehensive assessment.

+

How is user data protected?

We prioritize user privacy and data security. All health metrics and facial images are securely stored and transmitted using robust encryption methods. We comply with relevant data protection regulations to ensure that users' personal information is safeguarded at all times.

+

What are the future plans for this project?

Future developments for this project include integrating real-time video analysis to enhance behavioral assessments during gameplay. Additionally, we plan to transition the application to a Flask-based architecture for better scalability and integration with robust databases like SQLite, SQL, or MongoDB. We also aim to host the application on cloud platforms such as AWS or Google Cloud for improved reliability and accessibility.
